# Установка

Выполнить:
```
cd laravel
composer install
```

Задать настройки базы данных в .env файле

Выполнить миграции:
```
php artisan migrate
php artisan db:seed
```

Установка фронтенда:
*После установки автоматически будет запущена команда для компиляции фронтенда (npm run production)*
```
npm install
```
